# Data Availability

This repository contains code for MORSA (Morphology-Omics Representation via Structural Alignment). It does not redistribute raw third-party H&E whole-slide images, RNA-seq files, controlled-access data or the gated UNI checkpoint.

Raw public data should be obtained from the original repositories and projects described in the manuscript, including TCGA and external GDC/CPTAC resources. The author-generated processed results and analysis source data are deposited in Zenodo. The current release (v1.1.0) has DOI `10.5281/zenodo.21436404`; the Concept DOI for all versions is `10.5281/zenodo.21195955`.

The record provides cross-cancer experiment summaries, transcriptome-recovery
metrics, structural evidence, gene-level biological analyses, analytical-model
results, molecular-readout metrics and virtual-cartography source values. It
includes a data dictionary, experiment configuration, provenance notes and
SHA256 checksums.
Raw WSI and RNA files, UNI features, the gated UNI checkpoint and complete
trained checkpoints are not redistributed.

The author-generated processed files are released under CC BY 4.0. Original
third-party resources remain governed by their source terms.
